Services

I accept orders for...

Linux System Administration

I can set up, configure, and maintain Linux servers (Astra, RedOS, Debian) for stable and secure operation. This includes user management, package installation, and basic troubleshooting to keep your systems running smoothly.

Penetration Testing & Vulnerability Assessment

I perform basic penetration tests to identify security weaknesses in your networks and applications. After testing, I provide clear reports with actionable recommendations to improve your security posture.

IT Infrastructure Monitoring

I can deploy and configure monitoring systems like Prometheus and Grafana to track your server performance and security events. This provides visibility into system health and helps detect anomalies early.

Network Infrastructure Setup

I will configure and manage core network services like DHCP and DNS to ensure reliable connectivity. I can also set up VPNs and basic firewalls using Mikrotik equipment to secure your network perimeter.

Security Hardening & Compliance

I can help harden your systems by implementing security policies and access controls in Active Directory. I'll ensure your infrastructure meets basic compliance requirements through proper configuration and documentation.

Audio Production & Sound Engineering

I offer audio production services for podcasts, videos, and musical projects, ensuring professional sound quality. Using digital audio workstations and mixing techniques, I can clean up recordings and create polished final mixes.
